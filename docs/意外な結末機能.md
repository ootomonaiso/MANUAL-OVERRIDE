# 意外な結末機能（Issue #24）

## 概要

Issue #24: **「分岐の「意外な結末」が存在しない——全ルートが予定調和で終わる」** への対応です。

現在の分岐ツリーはすべて「選んだ方向にそのまま進む」設計になっており、プレイヤーの行動（プレイスタイル）は結果に影響していませんでした。この機能により、プレイヤーの実際のプレイスタイルが隠しルートや意外な結末を生み出します。

## 実装内容

### 1. プレイスタイル分岐（隠しルート）

プレイヤーの行動パターンを分析し、隠しジャンルへ誘導する機能です。

#### 検出可能なプレイスタイル

| スタイルID | 条件 | 誘導ジャンル |
|---|---|---|
| `low_jumper` | ジャンプレート < 1.5% | Stealth / Horror |
| `jump_spammer` | ジャンプレート > 7% | Platformer |
| `left_runner` | 左移動比率 > 1.3倍 | Idle |
| `collision_prone` | 衝突レート > 0.5% | Survival |
| `speed_demon` | 右移動レート > 6% | Racing |
| `sniper` | 射撃レート 0.5%〜2% | STG |

#### 動作フロー

```
ゲームプレイ → ActionStats 収集 → detectPlayStyle() → genreBonus 計算 → ベイズ尤度に加算 → 隠しジャンル誘導
```

#### ファイル

- `src/domain/playStyleDetector.ts` - プレイスタイル検出ロジック
- `src/game/sideScroller.ts` - 衝突カウント追加（`stats.collisions`）

### 2. 矛盾選択ルート

選択肢が「どちらも悪そうな」ものになり、矛盾が蓄積するにつれて説明書が崩壊していく機能です。

#### 矛盾カードの仕組み

```json
{
  "id": "c-contradict-detail",
  "label": "説明書をもっと詳しくする",
  "contradictionPair": "c-contradict-brief",
  "reactionMessage": "…えっ、詳しくするの？ でも前の選択は全然逆だったよ？"
}
```

矛盾ペアの両方が選択されると:
1. 矛盾レベルが 0.15 ずつ上昇
2. 反応メッセージがトースト表示
3. 説明書の関連テキストに取消化線
4. 矛盾レベル 0.6 到達でバッドエンド発動

#### 矛盾レベルの段階

| レベル | 深刻度 | 効果 |
|---|---|---|
| 0 | none | なし |
| 0〜0.3 | mild | 取消化線表示 |
| 0.3〜0.42 | moderate | 警告メッセージ + 説明書に滲み効果 |
| 0.42〜0.6 | severe | 強い警告 + 説明書に揺れ効果 |
| 0.6+ | broken | バッドエンド発動 |

#### ファイル

- `src/domain/contradictionTracker.ts` - 矛盾トラッキングロジック
- `src/data/cards/surprise-cards.json` - 矛盾カード定義（6ペア）
- `src/composables/useGameState.ts` - 矛盾状態管理

### 3. バッドエンド分岐

矛盾レベルが閾値を超えると、「説明書が壊れる」ルートが発動します。

#### 特徴

- ジャンルが `'glitch'` に強制変更
- エラー風演出（`ERROR: MANUAL_CORRUPTION`）
- 赤いグlichen アニメーション
- 通常のスコア表示・グレード表示を省略
- 独自のフレーバーテキスト

#### ファイル

- `src/data/genres/glitch.json` - glitch ジャンル定義
- `src/components/EndingPanel.vue` - バッドエンド演出

## 設定

### プレイスタイル閾値の調整

`src/domain/playStyleDetector.ts` の定数を調整:

```typescript
const JUMP_RATE_LOW = 0.015     // low_jumper 閾値
const JUMP_RATE_HIGH = 0.07     // jump_spammer 閾値
const LEFT_DOMINANCE_RATIO = 1.3  // left_runner 閾値
const COLLISION_RATE_HIGH = 0.005  // collision_prone 閾値
const SPEED_DEMON_RATE = 0.06  // speed_demon 閾値
```

### 矛盾システムの調整

`src/domain/contradictionTracker.ts` の定数を調整:

```typescript
const CONTRADICTION_INCREMENT = 0.15   // 1回あたりの増加量
const BAD_ENDING_THRESHOLD = 0.6       // バッドエンド発動閾値
const CONTRADICTION_WARNING_THRESHOLD = 0.3  // 警告表示閾値
```
