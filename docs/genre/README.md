# ジャンル別ドキュメント

各ジャンルの実装詳細をまとめたドキュメントを統括するディレクトリ。
ジャンル横断の収束システム・フィーチャーシステムの解説は親ディレクトリの
[genre-system.md](../genre-system.md) / [feature-system.md](../feature-system.md) を参照。

## ドキュメント一覧

| ジャンルID | ドキュメント | 概要 |
|-----------|-------------|------|
| `stg` | [stg-genre.md](./stg-genre.md) | 横スクロールシューティング（重力なし自由移動＋射撃） |
| `tetris` | [tetris-genre.md](./tetris-genre.md) | 落ち物パズル（10×20グリッド・7-bag・ライン消去） |

## 記載方針

各ジャンルドキュメントは以下を含むことを推奨する。

- **概要**: そのジャンルが何であるか・横スクロール原点からどう変容するか
- **アーキテクチャ**: 関連ファイル構成
- **ジャンル定義**: `src/data/genres/<id>.json` の主要項目
- **収束条件**: 収束に必要なパラメータと閾値・分岐パス
- **ゲーム仕様**: 操作・スコア・固有メカニクス
- **ビジュアル方針**: プラグインの描画方針
- **実装上の注意点**
